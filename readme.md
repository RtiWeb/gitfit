## Install

**Install with npm**

```sh
$ npm i -g gitfit
```

**Install with Yarn**

```sh
$ yarn global add gitfit
```

**Run with npx (without installing)**

```sh
$ npx gitfit
```


## Usage

```
	Usage:

		$ gitfit <command> [<options>]

	Options:

		-h, --help              Display help information
		-v, --version           Output Initify version

	Examples:

		$ gitfit init
		$ gitfit start <feature-name>
		$ gitfit finish <new-tag>
```
